## System requirements
* Ruby
* Node.js
* Yarn
* SQLite3
* [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli#download-and-install)

## Installation

```sh
make setup
make test # run tests
make lint # run linter
make check # run tests and linter
make start # run server http://localhost:3000
```
